# Claude Code Capabilities Reference

Claude Code is a CLI tool that enables AI agents to develop complete applications autonomously.

## File Operations

- **Read** - Read files with full project awareness
- **Write** - Create new files
- **Edit** - Modify existing files with precision
- **Glob** - Pattern-based file search
- **Grep** - Search across entire codebases
- **Bash** - Execute shell commands for builds, tests, deployments

## Autonomous Development

- **Agent** - Creates sub-agents for complex multi-step tasks
- **TaskCreate/TaskUpdate** - Manages structured task lists
- Executes commands and handles errors systematically
- Modifies existing codebases with context awareness

## Project Intelligence

- Understands project structure automatically
- Maintains context across large codebases
- Integrates with language servers (LSP) for:
  - Go-to-definition
  - Find references
  - Hover documentation
  - Code intelligence

## When to Use Claude Code

- Build web applications (frontend/backend)
- Create CLI tools or scripts
- Develop APIs and microservices
- Start new projects with structured methodology
- Modernize legacy systems iteratively
- Automate development workflows
- Refactor or maintain codebases
- Create and manage plugins/extensions

| Advantage | Description |
